# CSE 2050: Data Structures & Algorithms (OOP-Focus)
## Course Overview
This repository contains my work for CSE 2050: Data Structures and Object-Oriented Design with Professor Dossay Oryspayev

Introduction to fundamental data structures and algorithms. The emphasis is on understanding how to efficiently implement different data structures, communicate clearly about design decisions, and understand the relationships among implementations, design decisions, and the four pillars of object-oriented programming: abstraction, encapsulation, inheritance, and polymorphism.
 * https://catalog.uconn.edu/undergraduate/courses/cse/

### Topics:
- Python Basics (Python Review)
- OOP and Testing
- Running Time Analysis
- Linear Data Structures
- Recursion and Dynamic Programming
- Searching and Sorting
- Divide and Conquer
- Mapping and Hashing
- Trees
- Priority Queues and Heaps
- Graphs

### Review of Course
Professor: D. O. (Dossay Oryspayev) <br>
**Difficulty:** 4/10 <br>
**Course-load:** 3/10 <br> <br>
One homework, one set of practice questions to check knowledge, and one lab (able to do 3 days ahead of lab time if you want). <br>
The lab is a simple test of knowledge - almost always 10-30 minutes.
The homework is a deep test of knowledge, usually a puzzle - anywhere from 30 minutes to a few hours.
All labs, homework, and practice question modules are posted on Monday of every week. This makes it super easy to get ahead and use class time to refine knowledge rather than play catchup.
